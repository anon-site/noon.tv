This file documents the icons needed for PWA installation across different platforms.

- favicon.ico (16x16, 32x32, 48x48)

## Design Guidelines:
- Use the NOON TV branding colors (#ff6b35, #1a1a1a)
- Keep the design simple and recognizable at small sizes
- Use rounded corners for modern look
- Ensure good contrast for accessibility
- Test on both light and dark backgrounds

## Icon Generation Tools:
1. Online: favicon.io, pwa-asset-generator.com
2. CLI: pwa-asset-generator
3. Design tools: Photoshop, Figma, Canva

## Temporary Solution:
For testing, you can use online generators to create placeholder icons with:
- Background: #1a1a1a (dark gray)
- Text/Logo: "NOON" in #ff6b35 (orange)
- Rounded corners

## Installation:
Place all generated icons in the root directory alongside index.html and manifest.json.
